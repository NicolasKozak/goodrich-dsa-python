    \section{Prompt}

Show that $\sum^n_{i=1} i^2$ is $O(n^3)$.

    \pagebreak

    \section{Discussion}

$\sum^n_{i=1} i^2 \leq \sum^n_{i=1} n^2 = n n^2 = n^3$ for $n \geq 1$. Hence, $\sum^n_{i=1} i^2$ is $O(n^3)$.
